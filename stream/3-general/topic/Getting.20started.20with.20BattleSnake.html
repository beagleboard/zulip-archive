Hi all :smiley:

If you need a little help to get your BattleSnake up and controlling, the following step-summary may help? It's basically a condensed transcript of the introductory video tutorial:

BattleSnake (online game)

1.      Navigate to play.battlesnake.com

2.      Create a BattleSnake account (you can do so using your GitHub account)

3.      Setup your BattleSnake account.

Replit (online IDE)

4.      Navigate to replit.com and Create a Replit account.

5.      Setup account. Username. Name. Continue with Starter.

BattleSnake

6.      Getting Started > Starter Projects >   Choose a starter project
Note: Python is the most popular language.

GitHub (Battlesnake Python Starter Project)

7.      Click on ‘Run on Replit’

8.      Sign up if you’re not already logged in.

9.      Click on Remix.

10.  Provide name and description. Click on ‘Use Framework’

Replit is going to spin up a new replit for you. Click on ‘Run’

Note: File tree > Library >  main.py / server.py / readme etc

11.  Click on ‘+’on top bar to access ‘Advanced’ > Console

12.  Click on ‘Run’

13.  Customize your name and give the snake a fun color.

14.  Refresh or Stop / Start

15.  Open Developer > Copy Dev repl.co URL. This is the URL for your battle snake!

BattleSnake (creating your own snake)

16.  Create your own battle snake in the system -  play.battlesnake.com/account/battlesnakes.

Follow this link to create a new battle snake.

17.  Setup with standard engine region. Add the Replit URL (from above), platform etc

18.  Give your snake a description

19.  Save your battle snake

Should see a value to the left of ‘PING’ at top right

BattleSnake: it's GAME time!

20.  Click on the Create Game button (left hand side).

21.  Choose a map.

22.  My BattleSnakes > Add to game.

23.  Add a couple of other Bots?

24.  Click on ‘Start Game’

25.  Click Play

Note: Your Python snake has the basics to make them run a little. Now begin the process of making your snake better than random.
