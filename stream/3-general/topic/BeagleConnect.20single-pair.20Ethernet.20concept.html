Jason Kridner (Mar 27 2026 at 18:55):

I’ve started to talk publicly about my concept to use Baochip in a
BeagleConnect “SPE” (Single Pair Ethernet). Let me know if you are
interested in collaborating on the project. The basic idea is to make a
BeagleConnect Zepto form-factor board, but to have the JST QWIIC
connectors replaced by single-pair Ethernet with power-over-data-line
(PoDL) in a way that is compatible with BeaglePlay. There would need to
be a special cable to go from RJ11 to a 2-pin JST, but it would be nice
to work with as it is only 2 wires and should go several meters.
10Base-T1L would go a lot longer, but the 5V power source will limit the
wire length.
